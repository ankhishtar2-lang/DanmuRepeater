📝 项目简介
6657直播复读专用
🎯 核心功能
1. 智能弹幕捕获与加权选择
实时监听弹幕数据，基于 频次 × 长度加成 × 优先词加成 进行加权随机选择
趋势置顶：频次≥阈值的弹幕优先发送，响应刷屏热点
支持优先词和屏蔽词配置（支持正则表达式）
2. 动态调速（DPM）
基于每分钟弹幕量自动切换 疯狂/正常/佛系 模式
弹幕越密集，复读越快；弹幕稀薄时自动降低频率，避免刷屏
3. 多重去重
时间窗口去重（可配置）：窗口期内不重复发送同一条弹幕
历史队列去重（可配置）：避免长期复读少数弹幕
4. 高度可配置
弹幕长度偏好、发送间隔、去重参数、屏蔽词/优先词、高级筛选规则（length>10 / contains:哈哈 / regex:/^\d+$/）
主题定制：可自定义面板颜色、透明度、字体大小
🛠️ 技术架构
模块	实现
弹幕采集	MutationObserver + requestIdleCallback 批量处理
权重计算	增量更新，使用 requestIdleCallback 在空闲时计算
发送模拟	模拟完整用户交互（focus/input/click/keyboard），适配 React/Vue 框架
平台适配	动态选择器探测 + 启发式回退（联合选择器），自适应页面变化
数据持久化	GM 存储（配置、屏蔽词、优先词、主题、面板位置）
